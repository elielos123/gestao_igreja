# Manual do Usuário

## Login
Ao acessar o sistema, você será redirecionado para a tela de login. Insira suas credenciais para continuar.

## Dashboard
Visão geral financeira da igreja, com totais de entradas e saídas.

## Módulo Financeiro
- **Entradas**: Registro de dízimos e ofertas. Possui autocomplete para nomes de membros e congregações.
- **Saídas**: Registro de despesas (aluguel, luz, salários, etc).
- **Relatórios**: Filtros por data, congregação e tipo de movimento. Geram listas detalhadas e balanços mensais.

## Boas Práticas
1. **Sempre faça Logout** ao terminar de usar o sistema em computadores compartilhados.
2. **Backups**: Utilize a função de exportação de SQL periodicamente.
3. **Credenciais**: Nunca compartilhe sua senha administrativa.
